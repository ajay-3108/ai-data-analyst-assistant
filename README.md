# 📊 AI Data Analyst Assistant

An AI-powered analytics application that enables users to upload datasets and instantly generate business insights, visualizations, and interactive data exploration using Google Gemini AI.

## 🚀 Overview

This project combines traditional exploratory data analysis (EDA) with generative AI to help users understand datasets without writing code.

Users can upload CSV or Excel files and receive:

* Automated dataset profiling
* AI-generated business insights
* Data quality assessment
* Trend identification
* Interactive visualizations
* Natural language dataset querying

The goal is to make data analysis faster and more accessible for business users, analysts, and non-technical stakeholders.
