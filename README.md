# AI Data Analyst Assistant

## What this project does
Upload any CSV/Excel file and get:
- AI-generated insights (Gemini AI)
- Interactive charts (bar, scatter, histogram, heatmap)
- Dataset summary (EDA automation)
- Chat with your dataset

## Tech Stack
Python, Streamlit, Pandas, Plotly, Google Gemini AI

## How to run
pip install -r requirements.txt  
streamlit run app.py
